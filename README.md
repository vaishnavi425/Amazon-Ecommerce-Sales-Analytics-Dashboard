# Amazon E-Commerce Sales Analytics Dashboard
SQL and Power BI project for analyzing Amazon E-Commerce sales performance and customer insights.


## Project Overview

This project analyzes Amazon E-Commerce sales data using SQL and Power BI to uncover business insights related to revenue, customer behavior, order status, fulfilment performance, and geographic sales distribution.

## Tools Used

* MySQL
* Power BI
* SQL
* Data Analysis
* Data Visualization

## Dataset Information

The dataset contains Amazon sales transactions with information related to:

* Orders
* Revenue
* Quantity
* Categories
* States
* Cities
* Fulfilment
* B2B/B2C Customers

## SQL Analysis Performed

* Revenue by State
* Revenue by Category
* Order Status Analysis
* B2B vs B2C Analysis
* Fulfilment Performance Analysis
* Top Performing Locations

## Power BI Dashboard Pages

### 1. Sales Performance Dashboard

* Total Revenue
* Total Orders
* Total Quantity
* State-wise Analysis
* Revenue Trends
* Order Status Distribution

### 2. Customer Insights Dashboard

* Top 10 Cities by Revenue
* Revenue by Category
* Most Purchased Sizes
* Revenue by Sales Channel
* State-wise Revenue Map

### 3. Business Insights Dashboard

Key Findings:

* Maharashtra generated the highest revenue.
* Set category contributed the highest sales.
* B2C sales dominated total revenue.
* More than 63% of orders were successfully shipped.

Recommendations:

* Increase marketing efforts in top-performing states.
* Maintain inventory for high-selling categories.
* Improve logistics to reduce cancellations.
* Expand successful categories into additional regions.

## Project Outcome

This project demonstrates end-to-end data analysis skills including SQL querying, business analysis, KPI tracking, data visualization, and actionable insight generation using Power BI.
